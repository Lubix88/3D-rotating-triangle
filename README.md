# 3D rotating triangle

## Installing

**Python 3.9 or higher is required**
```
# Linux/macOS

git clone https://github.com/LubixDev/3D-rotating-triangle
pip3 install -r requirements.txt
python3 main.py

# Windows

git clone https://github.com/LubixDev/3D-rotating-triangle
pip install -r requirements.txt
python3 main.py
```
